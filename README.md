# Medicine Tracker Home

## Overview
The Medicine Tracker Home project is designed to help users easily track their medication intake schedules, dosages, and refill reminders. This application aims to enhance medication adherence and ensure timely intake of prescribed medicinal treatments.

## Features
- **User Authentication:** Secure login and registration for users.
- **Medication Management:** Add, edit, and delete medications.
- **Reminders:** Set up alerts for medication times and refills.
- **History Tracking:** View past medication intakes and adherence levels.
- **User-Friendly Interface:** A clean and straightforward UI for ease of use.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/adhome10/medicine-tracker-home.git
   cd medicine-tracker-home
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Setup the environment variables (create a .env file):
   ```bash
   PORT=your_port
   DATABASE_URL=your_database_url
   JWT_SECRET=your_jwt_secret
   ```
4. Start the server:
   ```bash
   npm run start
   ```

## Usage
- After starting the server, navigate to the application in your web browser.
- Register a new account or log in with existing credentials.
- Start adding your medications and set reminders as needed.
